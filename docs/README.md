# 🌾 AgriSwarm — Операционная Система для Умных Теплиц и Ферм


Добро пожаловать в официальную документацию проекта **AgriSwarm v4.0.4**!

AgriSwarm — это децентрализованная, отказоустойчивая прошивка (на базе ESP32 и FreeRTOS) для построения надежных сетей управления сельскохозяйственными объектами. Система не требует центрального сервера, работает в условиях жестких радиопомех и способна самовосстанавливаться при сбоях.

## 🌟 Ключевые возможности

*   **Offline-First:** Узлы общаются напрямую друг с другом (Wi-Fi Mesh / painlessMesh), минуя облако. Если пропадает интернет или отключается центральный сервер, теплица продолжает работать автономно.
*   **Zero-Alloc Architecture:** Ядро написано с запретом на динамическое выделение памяти в главном цикле, что гарантирует аптайм в несколько месяцев без фрагментации кучи и Kernel Panic.
*   **Бортовой самописец (BlackBox):** В случае аппаратного сбоя или отключения питания, система сохраняет последние логи в энергонезависимую RTC-память для последующего расследования (Post-Mortem).
*   **Умная Автоматизация:** Встроенный движок правил (`RuleEngine`) с кэшированием состояний O(1) и интерпретатор сложных логических цепочек с задержками (`AlgorithmScheduler`).
*   **Гарантированная доставка команд:** Защита критических команд (например, остановка насоса) от потери питания с помощью механизма Write-Ahead Logging (`CommandBackupSystem`).

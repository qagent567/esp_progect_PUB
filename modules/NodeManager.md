# 🖥️ Модуль NodeManager (src/NodeManager.cpp)

## 1. Обзор модуля и его роль в системе
`NodeManager` — это локальный диспетчер идентификации и ролевой модели. Он отвечает за определение того, "Кто я такой?" и базовый учет соседних узлов в сети.

**Основная роль:**
*   Управление локальным Node ID (генерируется аппаратно из eFuse MAC).
*   Управление ролью текущего узла (Sensor, Actuator, Gateway, Hybrid).
*   Управление уровнем доступа (`AccessLevel`), который определяет права узла в сети.
*   Ведение базового кэша `_nodes` — списка соседей и их времени последнего отклика (Heartbeat).

## 2. Ролевая модель и Уровни доступа (ACL Base)

В AgriSwarm узлы не равны между собой. `NodeManager` реализует фундамент для Access Control List (ACL).
При изменении роли узла (`setLocalNodeRole`), система автоматически назначает ему соответствующий уровень доступа (`_getAccessLevelForRole`):
*   `NODE_TYPE_SENSOR` -> `ACCESS_LEVEL_READONLY`. Узел-датчик имеет право только вещать данные. Если хакер перехватит контроль над ним, он не сможет отправить команду "Открыть все клапаны", так как его уровень доступа слишком низок.
*   `NODE_TYPE_ACTUATOR` -> `ACCESS_LEVEL_BASIC`.
*   `NODE_TYPE_HYBRID` -> `ACCESS_LEVEL_ADVANCED`.
*   `NODE_TYPE_GATEWAY` -> `ACCESS_LEVEL_ADMIN`. Узел, подключенный к Интернету или главному терминалу.

## 3. Мониторинг "Мертвых душ"
Метод `checkNodeLiveness()` работает в фоне (вызывается из `update`).
Если узел из словаря `_nodes` не обновлял свой `lastSeen` дольше `AGRI_NODE_TIMEOUT`, его статус меняется на `NODE_STATUS_DISCONNECTED`.
В целях экономии оперативной памяти, если неактивный узел не появляется очень долго (`AGRI_NODE_TIMEOUT * 3`), а размер словаря превышает 100 элементов (чтобы не допустить исчерпания RAM в огромных сетях), запускается `_removeInactiveNodes()`, и узел стирается из памяти.

## 4. Вывод
`NodeManager` — это легковесный паспортный стол устройства. Его главная заслуга — жесткая привязка ролей к привилегиям (`AccessLevel`), что закладывает основу для защиты Mesh-сети от внутренних нарушителей (компрометации отдельных периферийных узлов).

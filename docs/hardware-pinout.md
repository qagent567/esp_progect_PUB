# 📍 Распиновка и подключение (Hardware Pinout)

Чтобы правильно подключить датчик или реле к плате ESP32, нужно знать, какой контакт (пин) для чего предназначен. Использование неправильного пина может привести к тому, что плата не включится или датчик будет выдавать неверные данные.

---

## 🚦 Светофор контактов (Безопасность)

Не все контакты на плате одинаковы. Мы разделили их на три группы:

### 🟢 ЗЕЛЕНЫЕ (Безопасные)
*Используйте их в первую очередь. Они ни на что не влияют при запуске.*
*   **Пины:** `13, 14, 16, 17, 18, 19, 21, 22, 23, 25, 26, 27, 32, 33`
*   **Для чего:** Реле, кнопки, датчики температуры (DHT22), сервоприводы.

### 🟡 ЖЕЛТЫЕ (Осторожно)
*Эти пины могут выдавать сигналы при включении или имеют ограничения.*
*   **Пины `34, 35, 36, 39`:** Работают **ТОЛЬКО на вход**. К ним можно подключить датчик влажности почвы, но нельзя подключить реле или лампочку.
*   **Пины `0, 2, 5, 12, 15`:** Критичны для загрузки. Если к пину `0` или `2` подключить кнопку и зажать её при включении, плата уйдет в режим прошивки и не запустит вашу программу.

### 🔴 КРАСНЫЕ (Не трогать)
*   **Пины `1, 3`:** Это "рот" и "уши" платы для общения с компьютером (TX/RX). Если занять их датчиком, вы не сможете пользоваться терминалом CLI.
*   **Пины `6 - 11`:** Внутренняя память Flash. Подключение чего угодно к ним приведет к немедленному зависанию.

## 🗺️ Визуальная схема (ESP32 DevKit v1)

```text
                          +-------+
                      EN  |       |  3.3V  <-- Питание датчиков
     (Вход)  SENSOR_VP 36 |       |  GND   <-- Общий минус
     (Вход)  SENSOR_VN 39 |       |  D13   🟢 (Безопасный)
     (Вход)      IO34  34 |       |  D12   🟡 (BOOT)
     (Вход)      IO35  35 |       |  D14   🟢 (Безопасный)
   (Безопасный)  IO32  32 |  ESP  |  D27   🟢 (Безопасный)
   (Безопасный)  IO33  33 |  32   |  D26   🟢 (Безопасный)
   (Безопасный)  IO25  25 |       |  D25   🟢 (Безопасный)
   (Безопасный)  IO26  26 |       |  D33   🟢 (Безопасный)
   (Безопасный)  IO27  27 |       |  D32   🟢 (Безопасный)
   (Безопасный)  IO14  14 |       |  D35   🟡 (Только вход)
   (Безопасный)  IO12  12 |       |  D34   🟡 (Только вход)
   (Безопасный)  IO13  13 |       |  VN    (Вход питания 5В)
                      GND |       |  VP    (Вход питания 5В)
                          +-------+
```

## 🔌 Шпаргалка: Куда что втыкать?

| Если у вас... | Куда подключать (Пример) | Тип в pin_setup |
| :--- | :--- | :--- |
| **Реле (Помпа/Свет)** | Пин `25`, `26`, `27` | `RELAY` |
| **Кнопка** | Пин `32`, `33` | `DIGITAL_IN` |
| **Датчик DHT22** | Пин `4`, `14`, `16` | `DHT22` |
| **Датчик почвы (АЦП)** | Пин `34`, `35` | `ANALOG_IN` |
| **Сервопривод** | Пин `18`, `19` | `SERVO` |

---
💡 **Важно:** Всегда соединяйте контакт **GND** платы с минусовым проводом вашего датчика или модуля реле. Без общей "земли" система работать не будет.

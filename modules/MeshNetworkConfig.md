# ⚙️ Модуль MeshNetworkConfig

**Путь к файлам:**
- `include/MeshNetworkConfig.h`

## Обзор модуля
`MeshNetworkConfig` — это критически важный заголовочный файл, описывающий структуру `MeshNetworkConfig` и предоставляющий 5 захардкоженных **сетевых пресетов** (Presets) для адаптации параметров беспроводной Wi-Fi Mesh-сети (painlessMesh) под конкретные физические условия эксплуатации роя AgriSwarm.

## Структура конфигурации
Каждый узел хранит свою конфигурацию в энергонезависимой памяти (`/mesh_config.json` в LittleFS). Конфигурация занимает всего 32 байта памяти и содержит:
- **heartbeatStable / Normal / Search:** Таймауты обмена пакетами жизнеобеспечения (зависят от размера роя).
- **nodeTimeout:** Время, через которое узел считается "мертвым" (отключенным), если от него нет пингов.
- **connectionTimeout / reconnectInterval:** Настройки таймеров подключения к узлам.
- **pingInterval:** Частота отправки Keep-Alive пакетов.
- **channel:** Рабочий WiFi канал (1-13). Важно: все устройства в рое AgriSwarm **обязаны** использовать один и тот же канал для корректного автопостроения Wi-Fi Mesh-сети.
- **swarmEnabled:** Флаг глобального включения/отключения Mesh-сети.

## Доступные пресеты (Mesh Presets)

В файле определен массив `MESH_PRESETS`, содержащий заранее просчитанные профили. Пользователь может выбрать их через CLI (например, `mesh_preset eco`).

| Название | Назначение | Heartbeat | Node Timeout | Особенности |
|:---|:---|:---|:---|:---|
| **STABLE** `(0)` | Промышленный профиль | 30 сек | 2 минуты | Максимальная помехоустойчивость для заводов и цехов с толстыми стенами. Редкие пинги, большие таймауты. Узел не будет постоянно переподключаться из-за кратковременных радиопомех. Канал 1. |
| **BALANCED** `(1)` | По умолчанию | 15 сек | 1 минута | Баланс между скоростью реакции и стабильностью. Канал 6. |
| **FAST** `(2)` | Для разработки | 8 сек | 15 сек | Мгновенная реакция на отключение узла. Используется на столе разработчика. Канал 11. |
| **ECO** `(3)` | Энергосбережение | 60 сек | 3 минуты | Минимальное использование радиомодуля. Применяется для автономных датчиков на батарейках (в поле/лесу). Дает максимальную автономность. |
| **CUSTOM** `(4)` | Пользовательский | Ручной | Ручной | Загрузка полностью произвольных параметров из JSON. |

## Почему это вынесено в отдельный файл?
Выделение этих настроек в `MeshNetworkConfig.h` позволяет использовать O(1) доступ к параметрам в "горячем цикле" роутера, избегая постоянного чтения JSON файла при проверке таймаутов, что экономит процессорное время и защищает кучу (Heap) от фрагментации.

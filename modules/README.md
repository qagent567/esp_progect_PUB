# 🏗️ Технический анализ модулей AgriSwarm (Под капотом)

Данный раздел содержит глубокий технический анализ 38 ключевых программных модулей (классов C++) прошивки AgriSwarm. Описана логика работы алгоритмов, структуры данных, сетевое взаимодействие и механизмы оптимизации памяти (Zero-Allocation).

> [!IMPORTANT]
> В целях информационной безопасности и защиты интеллектуальной собственности от уязвимостей, исходный C++ код скрыт. Вся логика изложена концептуально, в виде схем, таблиц и текстовых описаний алгоритмов.

## 🗺️ Список системных модулей

| Модуль (Класс / Файл) | Краткое описание роли подсистемы |
| :--- | :--- |
| **[SystemOrchestrator](./SystemOrchestrator.md)** | Главный фасад жизненного цикла прошивки, запуск менеджеров, кооперативный цикл `loop()`. |
| **[PinManager](./PinManager.md)** | Слой аппаратной абстракции (HAL), управление GPIO, драйверная модель датчиков/серво/реле. |
| **[EventManager](./EventManager.md)** | Асинхронная шина событий (Event EventBus) для слабой связанности внутренних компонентов. |
| **[RuleEngine](./RuleEngine.md)** | Локальный движок правил (рефлекторная автоматика) с оптимизацией через O(1) хэширование. |
| **[AlgorithmScheduler](./AlgorithmScheduler.md)** | Планировщик сложных временных сценариев автоматизации (например, фазовый полив по расписанию). |
| **[SmartMeshManager](./SmartMeshManager.md)** | Интеллектуальный Mesh-роутинг, балансировка нагрузки (Load Balancing) и резервные узлы (Backup Hosts). |
| **[AgriNetworkManager](./AgriNetworkManager.md)** | Сетевой стек для передачи пакетов в Wi-Fi Mesh с поддержкой гибридного WiFi. |
| **[TrustedNodeManager](./TrustedNodeManager.md)** | Система безопасности и контроля списков доступа (ACL) по MAC-адресам узлов в сети. |
| **[BlackBoxManager](./BlackBoxManager.md)** | Бортовой регистратор сбоев (самописец) в RTC RAM с поддержкой износостойкого гибридного дампа во Flash. |
| **[CoreStabilityManager](./CoreStabilityManager.md)** | Защита от зависаний задач через аппаратные сторожевые таймеры (Watchdog) FreeRTOS. |
| **[SelfReflectionSystem](./SelfReflectionSystem.md)** | Профилактическая самодиагностика, сборка мусора и восстановление маршрутов в реальном времени. |
| **[CommandExecutor](./CommandExecutor.md)** | Асинхронный CLI-интерфейс в терминале с автодополнением (Tab) и историей команд. |
| **[SensorMonitor](./SensorMonitor.md)** | Накопление долгосрочной статистики (мин/макс) и расчет экспоненциального скользящего среднего (EMA). |
| **[SensorPublisher](./SensorPublisher.md)** | Публикация локальной телеметрии и подписка на удаленные датчики в Mesh-сети. |
| **[ActuatorPublisher](./ActuatorPublisher.md)** | Публикация актуаторов и отправка команд управления с гарантированной доставкой (ACK). |
| **[CommandBackupSystem](./CommandBackupSystem.md)** | Резервное копирование критических команд во Flash для защиты от внезапного пропадания питания. |
| **[ConnectionLossDetector](./ConnectionLossDetector.md)** | Контроль стабильности соединений и перевод критических исполнительных механизмов в безопасный режим (Fail-Safe). |
| **[EventBuffer](./EventBuffer.md)** | Фильтрация дребезга контактов и накопление сырых физических сигналов с входов. |
| **[HashUtils](./HashUtils.md)** | Сверхбыстрое хэширование FNV-1a, выполняемое в `constexpr` на этапе компиляции. |
| **[HostManager](./HostManager.md)** | Менеджер аварийной точки доступа (AP Mode) для локальной настройки при изоляции узла. |
| **[Logger](./Logger.md)** | Потоковый форматированный логгер с защитой от переполнения стека и авто-перехватом ошибок в BlackBox. |
| **[MessageBuffer](./MessageBuffer.md)** | Циклический кольцевой буфер текстовых сообщений логов в ОЗУ для команды `history`. |
| **[MessageCRC](./MessageCRC.md)** | Защита от помех радиоэфира через быстрое табличное вычисление контрольной суммы CRC16-CCITT. |
| **[MessageQueueManager](./MessageQueueManager.md)** | Базовый диспетчер распределения очередей пакетов в сетевом стеке. |
| **[MessageRouter](./MessageRouter.md)** | Фильтрация, валидация сигнатуры и диспетчеризация входящих сетевых пакетов. |
| **[NetworkManager](./NetworkManager.md)** | Низкоуровневый интерфейс взаимодействия с физическим приемопередатчиком Wi-Fi. |
| **[NodeManager](./NodeManager.md)** | Локальная таблица и метрики работоспособности соседних физических узлов роя. |
| **[NotificationSystem](./NotificationSystem.md)** | Рендеринг и вывод критических алертов поверх CLI-интерфейса Serial консоли. |
| **[PriorityMessageQueue](./PriorityMessageQueue.md)** | Zero-Alloc очередь с вытеснением низкоприоритетных пакетов при нехватке памяти (Max-Heap). |
| **[SafeCommandParser](./SafeCommandParser.md)** | Безопасная токенизация команд без риска утечек памяти или порчи исходной строки. |
| **[SafeMath](./SafeMath.md)** | Защищенные математические вычисления с проверками на переполнение типа и деление на ноль. |
| **[SafeMemory](./SafeMemory.md)** | Функции безопасной работы с массивами байт и копирования строк через `strncpy`. |
| **[Scheduler](./Scheduler.md)** | Легковесный планировщик некритичных периодических задач главного цикла. |
| **[SensorConnectionMonitor](./SensorConnectionMonitor.md)** | Контроль стабильности шин данных и питания подключенных датчиков. |
| **[SensorMonitor](./SensorMonitor.md)** | Мониторинг показаний датчиков во времени, фильтрация шумов измерений. |
| **[SystemDiagnostics](./SystemDiagnostics.md)** | Сбор данных о напряжении питания, температуре кристалла ESP32 и частоте процессора. |
| **[SystemMonitor](./SystemMonitor.md)** | Мониторинг времени цикла loop для выявления блокирующих вызовов в драйверах. |
| **[CommandExecutor](./CommandExecutor.md)** | Выполнение парсинга и маршрутизация CLI команд от пользователя. |
| **[Constants](./Constants.md)** | Файл глобальных системных констант, таймаутов, путей к файлам конфигурации. |
| **[Structures](./Structures.md)** | Файл глобальных типов, перечислений (enum) и структур (struct) конфигурации. |
| **[MeshOptimizer](./MeshOptimizer.md)** | Управление динамической оптимизацией сети под скорость, надежность или энергосбережение. |
| **[MeshReliabilityManager](./MeshReliabilityManager.md)** | Контроль стабильности топологии, QoS, альтернативные маршруты и ретрансляция пакетов. |
| **[MeshPerformanceMonitor](./MeshPerformanceMonitor.md)** | Потокобезопасный сбор метрик задержек, пропускной способности и качества радиоканалов. |
| **[MeshDeliveryTracker](./MeshDeliveryTracker.md)** | Отслеживание доставки пакетов по транзакционным ID, подтверждения (ACK/NACK) и авто-повторы. |
| **[AutoConnectionManager](./AutoConnectionManager.md)** | Легковесный автоподключатель к сети с экспоненциальным бэкоффом и случайным джиттером. |
| **[MeshStatisticsManager](./MeshStatisticsManager.md)** | Сбор и износостойкое сохранение во Flash статистики линков, RSSI и стабильности узлов. |
| **[PingManager](./PingManager.md)** | Контроль доступности узлов (Keep-Alive), измерение RTT и непрерывный Windows-style пинг. |
| **[MeshNetworkConfig](./MeshNetworkConfig.md)** | Конфигурационные пресеты (STABLE, ECO, FAST) и профили энергопотребления Mesh-сети. |
| **[MqttManager](./MqttManager.md)** | Интеграция с внешними MQTT-брокерами для облачного мониторинга и телеметрии. |
| **[HardwareDrivers](./HardwareDrivers.md)** | Драйверы периферии (DHT, Analog, Servo, Relay) на базе паттерна Стратегия. |
| **[CommandHandlers](./CommandHandlers.md)** | Маршрутизация CLI-команд через паттерн "Цепочка обязанностей". |
| **[LogPauseManager](./LogPauseManager.md)** | Автоматическая пауза фонового логгера при вводе CLI-команд пользователем. |
| **[PinCommandHelper](./PinCommandHelper.md)** | Строгая валидация GPIO и парсинг команд настройки аппаратных пинов. |

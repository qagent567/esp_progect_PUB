# 📦 archive/ — Архив черновиков документации

Эта папка содержит **черновые и устаревшие версии** технических описаний, которые использовались на ранних этапах разработки проекта. Они не предназначены для конечного пользователя и хранятся здесь исключительно в архивных целях.

> [!NOTE]
> **Не читайте эти файлы для изучения проекта.** Актуальная документация находится в корне репозитория и в разделе `docs/`.

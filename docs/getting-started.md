# 🎓 Быстрый старт (Getting Started)

В этом уроке мы пройдем путь от чистой платы ESP32 до работающего узла AgriSwarm, который читает данные с датчика температуры.

**Цель:** Получить базовое понимание работы с консолью (CLI) AgriSwarm за 10 минут.

## Шаг 1: Прошивка и подключение

1. Подключите ESP32 к компьютеру по USB.
2. Прошейте плату одним из способов:
   * **Вариант А (Основной):** Запустите графический прошивальщик `flash_firmware.exe` (или `flash_firmware.py`, если у вас установлен Python) из папки [`firmware/`](../firmware/README.md). Он автоматически загрузит готовую прошивку v4.0.4.
   * **Вариант Б (Ручной):** Загрузите готовую прошивку вручную с помощью утилиты `esptool.py` (подробные адреса разделов и параметры загрузки указаны в руководстве в папке `firmware/`).
3. Откройте Serial Monitor со скоростью **115200 бод** и нажмите кнопку сброса (RST/EN) на плате.
4. Вы должны увидеть ASCII-логотип AgriSwarm и приглашение командной строки:
   ```text
   agriswarm$ 
   ```

*💡 Подсказка: Используйте клавишу `Tab` для автодополнения команд и стрелки Вверх/Вниз для навигации по истории.*

## Шаг 2: Установка роли узла

По умолчанию "голый" узел не имеет определенных прав в сети. Назначим ему роль "Гибридного" узла (может и читать датчики, и управлять реле).

Введите команду:
```bash
node_role hybrid
```
Система автоматически присвоит узлу уровень доступа `Advanced` и сохранит конфигурацию в энергонезависимую память.

## Шаг 3: Настройка датчика температуры

Представим, что к пину GPIO 4 подключен датчик DHT22.
Используйте команду `pin_setup` (формат: `pin_setup <имя> <тип> <gpio>`):

```bash
pin_setup temp_greenhouse DHT22 4
```
Вывод системы подтвердит регистрацию датчика в модуле `PinManager`.

## Шаг 4: Проверка данных

Чтобы увидеть, что датчик действительно работает и считывает температуру, запросите диагностику пина:

```bash
pin_info temp_greenhouse
```
Если вы хотите посмотреть список всех зарегистрированных интерфейсов (входов и выходов):
```bash
pin_list detailed
```

## Шаг 5: Мониторинг Mesh-сети

Если у вас есть вторая прошитая плата ESP32, просто включите её в соседней комнате. Сетевой слой `AgriNetworkManager` найдет её автоматически (через Wi-Fi сканирование и Wi-Fi Mesh фреймы) в течение 5-10 секунд.

Чтобы посмотреть, какие узлы видит ваше устройство, введите:
```bash
node_list
```
Для просмотра графического дерева локальной сети (включая датчики и актуаторы) используйте:
```bash
pin_tree
```
